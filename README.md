# MirrorX Backend

Deployed API backend for MirrorX. Routes:
- `/` – Health check
- `/rpc-list` – Raw Solana RPC list
- `/rpc-status` – Live status from all endpoints
- `/run-job`, `/score-history`, `/live-score`, `/scores` – Data mocks
